PAUSE GAME

task 1: The secret message

Objective: The team needs to decode the first encrypted message, which contains crucial information about the location of a secret meeting between enemy agents. This will give them the next step in the mission.

Description:
The game begins as the team is handed an Encrypted Envelope. Inside, they find a piece of paper with a message that appears to be a jumble of letters, numbers, and symbols:

"Vxfo fq tlr! Mfu axq rzj qb yzqxo bmtf lqpc?"

The Tech Specialist has a tool that can help them decrypt part of the message, but it requires a password—and only by solving an initial puzzle can they obtain it.

Clue:
On the desk, they find a Confidential File with a small puzzle in it. The file contains a list of names, each followed by a number:

James – 6
Amelia – 4
Victor – 2
Eleanor – 8
The players must figure out that the numbers correspond to the number of letters in each person’s name. They need to extract the first letters of each name in the order given (J, A, V, E), which spells "JAVE". This is the password to unlock the decryption tool.

Once decrypted, the message reads: "Find the hidden room in the west wing. The key is in the safe."

Next Step:
With this newly decoded information, the team can now proceed to the West Wing in the game space, where they must find the hidden room and retrieve the key from the safe. The clock is ticking, and they must stay focused.
